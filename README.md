# 🌤️ Weather PWA

[![React](https://img.shields.io/badge/React-18.2.0-blue?logo=react)](https://reactjs.org/)
[![PWA](https://img.shields.io/badge/PWA-ready-brightgreen?logo=pwa)](https://web.dev/progressive-web-apps/)
[![OpenWeatherMap](https://img.shields.io/badge/API-OpenWeatherMap-orange?logo=openweathermap)](https://openweathermap.org/)

A progressive web application that displays weather information for any city. Built with React and OpenWeatherMap API.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Display temperature in Celsius
- 🏙️ Show city information and weather conditions
- 🌐 Works offline with service worker
- 📱 Installable as a mobile app (PWA)

## 🚀 Quick Start

1. Clone this repository
```bash
git clone https://github.com/EvgenyEysner/weather-pwa.git
```

2. Install dependencies
```bash
npm install
```

3. Replace with your OpenWeatherMap API key `fetchWeather.js`

4. Start the development server
```bash
npm start
```

## 🛠️ Built With

- React.js
